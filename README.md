# Neighborhood-map
This is a single page application featuring a neighborhood-map of favourite restaurants built using knockout.js,google apis and wiki apis.

# Skills
  - Javascript
  - JQuery
  - Knockout.js
  - Google Map API
  - wiki API
  - Bootstrap
  - CSS3
  - HTML5
# Getting Started
  - Download or clone the project.
  - open index.html.
